HOW TO UPDATE THE LIVE SITE
1. Open the atlas-bears-math-hub repository.
2. Choose Add file > Upload files.
3. Drag all Version 2 files and the assets folder onto the upload page.
4. GitHub will warn that some files already exist. This is expected.
5. Commit the changes.
6. Wait 1–3 minutes and refresh the live site.
